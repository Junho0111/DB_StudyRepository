### 제약 조건 (Constraints)

데이터의 무결성(정확성)을 유지하기 위한 규칙입니다.

| 제약 조건 | 의미 | 설명 |
| :--- | :--- | :--- |
| **NOT NULL** | 필수 입력 | 값이 비어있는 것을 허용하지 않음. |
| **UNIQUE** | 중복 불가 | 테이블 내에서 고유한 값이어야 함. |
| **PRIMARY KEY** | 기본 키 | 행을 식별하는 대표 키 (`NOT NULL` + `UNIQUE`). |
| **FOREIGN KEY** | 외래 키 | 다른 테이블과 연결 (참조 무결성 유지). |
| **DEFAULT** | 기본값 | 값 미입력 시 자동으로 들어갈 값 설정. |
| **CHECK** | 조건 검사 | 특정 조건(예: 나이 > 18)을 만족해야 함. |
